# Youtube API Search

#### This mini project uses the Youtube API and dynamically searches for videos when you type something in the search bar

#### How to start: 
     1.Install NodeJs
     2.Run commands:
     * 'npm install' in the project root directory
     * 'npm start' in the project root directory
     3. Open 'localhost:8080'
     
     Note: Webpack will build automatically after every save
